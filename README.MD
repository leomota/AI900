# Roadmap de Certificação AI-900

## Introdução
Bem-vindo ao roadmap de estudo para a certificação AI-900, que abrange os principais tópicos do exame.

## Tópicos Principais
- AI Workloads
- Fundamental Principles of Machine Learning
- Computer Vision Workloads
